# todo-webapp
 Eine Web-App zur Verwaltung einer To-Do-Liste

###Funktionen
- Anlegen von To-Do-Listen-Einträgen
- Bearbeiten von Einträgen
- Löschen von Einträgen
### Verwendete Techniken/Frameworks

- .NET 8.0
    - ASP.NET Core

- Angular 18.0.1
    - Angular CLI 18.0.2
    - Angular Material 18.0.1

- Node 20.11.1
    - npm 10.2.4
